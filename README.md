## 차트 컴포넌트 설명

| 파일명             | 설명                                                              |
| ------------------ | ----------------------------------------------------------------- |
| `BarChartList.jsx` | 모든 차트 컴포넌트, 차트 옵션 설정도 있음                         |
| `RechartBar.jsx`   | 분리된 Rechart 컴포넌트                                           |
| `data.js`          | 각 차트에 들어가는 샘플 데이터 (해당 형태로 데이터가 들어가야 함) |
